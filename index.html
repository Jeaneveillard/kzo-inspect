Ne pas ouvrir le fichier .html directement

KZO Inspect doit être lancé avec le serveur local (double-clic sur le fichier ci-dessous). Si vous ouvrez seulement index.html, le navigateur bloque l'application et vous ne voyez que le logo.

  1. Dans le dossier KZO Inspect sur le Bureau
  2. Double-cliquez sur Lancer KZO Inspect.bat (Windows) ou .command (Mac)
  3. Le navigateur s'ouvrira sur http://127.0.0.1:8775

Laissez la fenêtre Terminal/Console ouverte pendant que vous utilisez l'app.
